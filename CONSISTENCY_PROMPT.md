# Onboarding Consistency Prompt

Use this prompt to ensure new UniDoxia onboarding screens match the existing modern SaaS aesthetic. Keep it separate from other design prompts when generating visuals or UI concepts.

> Use the same visual style as the previous screens: modern SaaS UI, rounded cards, soft shadows, clean spacing, and subtle UniDoxia brand colors. Maintain a professional, premium, minimal aesthetic across all onboarding screens.
